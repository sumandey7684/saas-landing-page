# SaaS Landing Page

A modern and responsive landing page template for SaaS businesses, built with Next.js and Tailwind CSS.

## Overview

This project provides a clean and professional landing page template designed to showcase SaaS products and services. It's built using Next.js for server-side rendering and optimized performance, along with Tailwind CSS for rapid and customizable styling.

## Features

*   **Modern Design:** A visually appealing and user-friendly design.
*   **Responsive Layout:** Fully responsive design that adapts to different screen sizes.
*   **Next.js:** Utilizes Next.js for server-side rendering and improved performance.
*   **Tailwind CSS:** Uses Tailwind CSS for easy and customizable styling.
*   **Framer Motion:** Includes smooth animations and transitions using Framer Motion.
*   **Lenis:** Smooth scrolling experience with Lenis.

## Technologies Used

*   [Next.js](https://nextjs.org/) - React framework for building performant web applications.
*   [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework for rapid UI development.
*   [clsx](https://github.com/lukeed/clsx) - A tiny utility for constructing `className` strings conditionally.
*   [Framer Motion](https://www.framer.com/motion/) - A production-ready motion library for React.
*   [Lenis](https://github.com/studio-freight/lenis) - Smooth scroll-behavior with Ленис.
*   [TypeScript](https://www.typescriptlang.org/) - Superset of JavaScript which adds static typing.

## Installation

Follow these steps to set up the project locally:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/sumandey7684/saas-landing-page.git
    cd saas-landing-page
    ```

2.  **Install dependencies:**

    Using npm:

    ```bash
    npm install
    ```

    or using pnpm:

    ```bash
    pnpm install
    ```

3.  **Set up environment variables:**

    Create a `.env.local` file in the root directory and add any necessary environment variables.  Since this is a landing page, you might not need any environment variables to start, but this is where you would put them.

4.  **Run the development server:**

    Using npm:

    ```bash
    npm run dev
    ```

    or using pnpm:

    ```bash
    pnpm dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Usage

The landing page is designed to be easily customizable. You can modify the content, styles, and components to fit your specific SaaS product or service.

*   **Modify Content:** Edit the text and images in the `src/` directory to reflect your brand and messaging.
*   **Customize Styles:** Use Tailwind CSS classes to adjust the appearance of elements.  The `tailwind.config.ts` file is where you can customize the theme.
*   **Add Components:** Create new React components in the `src/components/` directory and import them into your pages.
*   **Update Sections:** Modify the sections in the `src/sections/` directory to match your content requirements.
